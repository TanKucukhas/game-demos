# Before the Thaw — The First Hunt

A playable vertical slice of the "systemic parenting adventure" polar bear game.
You are a mother polar bear, starved after four months in the den, with two cubs
who learn by watching you. Feed them, teach them, and get everyone through the
storm — across ice that doesn't always hold your weight.

## Run it

```bash
npm install
npm run dev        # → http://localhost:5173
```

The game is fully playable out of the box with procedural models and synthesized
audio — no API keys required.

## Generated assets (Meshy + ElevenLabs)

Real 3D models and cinematic audio are generated by API and dropped into
`public/models/` and `public/audio/`. The game detects them automatically on
next reload — no code changes.

```bash
cp .env.example .env   # then paste your keys into .env
npm run assets         # generates everything (Meshy takes ~5-15 min per model)
npm run assets:models  # Meshy only:  mother bear, cub, seal GLBs
npm run assets:audio   # ElevenLabs only: wind, calls, ice cracks, narration
```

- **Meshy** (`MESHY_API_KEY`): text-to-3D, preview → refine → GLB download.
- **ElevenLabs** (`ELEVENLABS_API_KEY`): sound-generation API for SFX, TTS for
  the documentary narrator (`ELEVENLABS_VOICE_ID` optional).

Already-generated files are skipped, so the script is safe to re-run.

## How to play

| Key | Action |
| --- | ------ |
| WASD | move / turn |
| Shift | run (burns energy fast) |
| C | toggle stalk stance (quiet, slow, low) |
| Space | pounce / pull a cub from the water |
| Q | call cubs back |
| E | tell cubs to stay / follow |
| F | nurse cubs (costs your energy) / hold at carcass or whale to feed |
| R | rest — shelter behind a pressure ridge to sit out the storm |
| G | play with the cubs — builds bond, improves obedience |

### The systems

- **Wind & scent** — the seal smells you if it is downwind of you. Watch the
  compass; approach so the wind blows *from* the seal *toward* you. The wind
  shifts every ~45 seconds.
- **Noise** — running near the hole spooks the seal; stalking is nearly silent.
- **Thin ice** — dark patches will not hold your weight. Break one and that
  crossing is gone for the whole run. Cubs are light enough to cross — but a
  broken patch is open water, and cubs drown in ~20 seconds.
- **Teaching** — a cub that watches a kill from within 25 m gains hunting
  knowledge, which also makes it obey commands more reliably. Cubs told to
  *stay* are safe from the hunt — and learn nothing.
- **The storm** — 6 minutes. Fed cubs + rebuilt reserves = the family endures.
- **The rival male** — a kill puts blood on the wind; ~40 s later a male twice
  your size arrives. Stand your ground (Space, energy-based contest) or retreat.
- **The whale carcass** — west across the open-water lead, guarded by two
  scavenger males. Charge them, or stalk in downwind and steal from the far side.
- **The lead** — open water with a south-running current. Cubs can swim ~11 s
  before exhaustion; the current drags everyone off course.
- **Rest & play** — shelter (R) pauses the drain and warms the cubs; play (G)
  builds bond that permanently improves obedience this run.
- **Survival Legacy** — if a cub survives, continue the bloodline: the next
  winter's cubs inherit hunting instinct and trust from this run.

### The cubs

- **Siku** — curious, fast, disobedient. Wanders toward thin ice and pressure
  ridges. Learns hunting quickly if allowed to watch.
- **Nukka** — obedient, slower, stays close. Will hold a *stay* command.
